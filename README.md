# MonkeMessenger

A Messenger for Gorilla Tag.

Made for the 2026 MonkeMod Modding Contest

Requires [GorillaInfoWatch](https://github.com/developer9998/GorillaInfoWatch)

Privacy Notice:

All personal information (such as your password whenever you create an account) is hashed, and cannot be accessed by anyone, not even myself.

Data is able to be deleted at request.
